Detects ASCII card data from MMD1100 output

test Firmware(with password)

Testing Procedure

1. Download the test firmware and use the LPU237 Ganymede MSR with the mapper for forced installation.
2. Set the MSR interface to USB keyboard mode.
3. Run Notepad on Windows 10 and swipe the test card.
4. If there are no reading errors, the MMD1100 encoder output for all tracks containing data will be displayed in hexadecimal, regardless of the mapper settings.
  - Before displaying each track value, two "enter" keystrokes are output.
  - After displaying each track value, one "enter" keystroke is output.
5. After the MMD1100 encoder output, the normal operating mode displays card data according to the mapper settings.
6. Copy the MMD1100 encoder output value for the card track you want to analyze and paste it into the **Encoder Data** input field.
7. Copy the ASCII string of the card track you want to analyze and paste it into the **ASCII Data** input field.
8. Click the **Generate ISO Data** button.
9. When a prompt appears, enter `6` for track 1 analysis or `4` for track 2 or track 3 analysis.
10. In the displayed results, if numbers are shown under **The positions:**, it indicates that the MMD1100 encoder output contains a pattern representing the card's ASCII string.
